This page demonstrates JavaScript's fetch() function. Fetch uses JavaScript's async/await interface. This version also has an associated node.js and express.js server.

It works by making an HTTP request in a Promise, then awaiting a response. A successful response returns a Response object. You usually convert the response to either text or JSON, and then use that result in your script. You might also need to catch an error if there is one, or you might need to set some parameters, like the headers of your HTTP request, or whether or not to permit CORS requests, and so forth. You can see examples of these in this page's script.

This page and server is also available at Glitch.com

For more info, see MDN's documentation on using fetch.
